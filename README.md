## Overview
Mark# is a simple, modern, and lightweight Markdown editor built with C# and WPF.

## Features
- **Tabbed Document Interface:** Have multiple files open at once, each in its own tab.
- **Live Preview:** A split-screen view with a real-time HTML preview that updates as you type.
- **Rich Text Formatting Toolbar:** Buttons for Bold (Ctrl+B), Italic (Ctrl+I), Links (Ctrk+K), Strikethrough, Code, and Blockquotes.
- **File Operations:** Full support for Open (Ctrl+O), Save (Ctrl+S), and Save As (Ctrl+Shift+S).
- **Export to HTML:** (Ctrl+Shift+E) Saves a self-contained .html file with all styles.
- **Print to PDF:** (Ctrl+P) "Prints" the document to a clean .pdf file.
- **Unsaved Changes Tracking:** The window title shows an * when your file has unsaved changes.
- **Status Bar:** Clean status bar with word count, theme toggle, and scroll sync toggle.
- **Scroll Syncing:** A toggleable feature that syncs the scroll position between the editor and preview panes.
- **Theme Support:** Toggle between light and dark modes, with the preview pane adapting automatically.
